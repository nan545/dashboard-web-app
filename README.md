Simple Business Dashboard Web App
Project Overview

This project is an intermediate-level web development assessment designed to evaluate practical frontend skills. The application is a responsive business dashboard that allows users to manage a list of clients and view basic statistics.

The focus is on functionality, code quality, UI/UX, and proper state management.
Tech Stack

HTML5 (Semantic HTML)

CSS3 (Flexbox / Grid)

JavaScript (ES6+)

Optional: React / Framework of choice

LocalStorage for data persistence
